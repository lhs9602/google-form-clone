# 구글 설문지


배포링크:https://lhs9602.github.io/google-form-clone/


![2023-10-18 15 49 38](https://github.com/lhs9602/google-form-clone/assets/34961388/b6e4ac36-dc90-462e-a5db-de4f1757ab8d)


## 기능 구현
- 설문지 제목 추가, 편집
- 질문 추가
- 질문 타입
    - 단답형
    - 장문형
    - 객관식 질문
    - 체크박스
    - 드롭다운
- 질문 복사 
- 질문 삭제 
- 필수 옵션 설정 
- 미리 보기
- 드래그로 질문과 옵션 순서변경
- 새로고침 시 데이터 유지
